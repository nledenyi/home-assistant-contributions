# Unified-look section cards

Dashboard pattern implemented via the [`ha-section-card`](../../skills/ha-section-card/) Claude Code skill. Each section is a Bubble-Card + card-mod + tile + entities composite over a translucent section background, sharing four reusable Bubble-Card modules for consistent look:

- `unified-look-hero` - transparent wrappers + typography bump for hero state cards
- `unified-look-separator` - softened divider line
- `unified-look-control` - translucent pill for sliders
- `unified-look-control-flat` - pill-less variant for presets and selects

Full module YAML is in the skill's `SKILL.md` appendix (see [`skills/ha-section-card/SKILL.md`](../../skills/ha-section-card/SKILL.md)).
